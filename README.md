# Enterprise AI Quality Framework

An enterprise-ready AI Quality Assessment Framework for evaluating Generative AI (GenAI) and Retrieval-Augmented Generation (RAG) applications.

---

## Overview

Enterprise AI Quality Framework helps organizations evaluate the quality of AI applications by measuring response quality, confidence, latency, and evaluation status.

The framework provides:

- Interactive Dashboard
- Executive Report
- Detailed Evaluation Report
- JSON-based reporting
- Configurable evaluation engine
- Template-driven report generation

---

## Key Features

- Configurable Framework
- Mock Evaluation Engine
- Dataset Driven Evaluation
- Quality Metrics Engine
