Enterprise GenAI RAG
Quality Assessment Framework

Evaluate • Validate • Trust Enterprise AI Responses

A portfolio project demonstrating how enterprise Retrieval-Augmented Generation (RAG) applications can be evaluated using AI Quality Engineering principles. The framework measures response quality through accuracy, grounding, relevance and hallucination detection using a sample enterprise knowledge base.
